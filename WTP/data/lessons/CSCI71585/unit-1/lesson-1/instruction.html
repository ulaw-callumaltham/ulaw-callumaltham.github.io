In this lesson, you will learn about the basic structure of a web page and how to use HTML tags to create a well-organized layout.


Understanding HTML Structure

HTML (HyperText Markup Language) is the standard language used to create web pages. It consists of various tags that define the structure and content of a web page. Understanding the basic structure of an HTML document is essential for building web pages effectively.

At its core, an HTML document is made up of several key elements:

  • <!DOCTYPE html>: This declaration defines the document type and version of HTML being used.
  • <html>: This is the root element that wraps all the content of the web page.
  • <head>: This section contains meta-information about the document, such as the title, character encoding, and links to stylesheets or scripts.
  • <body>: This is where the visible content of the web page goes, including text, images, links, and other media.

We use these elements to form the core structure of an HTML document. All HTML pages must include these elements to be valid.

You can see how they work together in the example below:

<!DOCTYPE html>
<html>
<head>
    <title>My First Web Page</title>
</head>
<body>
    <h1>Welcome to My Web Page</h1>
    <p>This is a paragraph of text on my first web page.</p>
    <img src="https://example.com/image.jpg" alt="Example Image">
    <a href="https://www.example.com">Visit Example.com</a>
</body>
</html>

In this example, you can see the basic structure of an HTML document with a doctype declaration, html element, head section, and body section (do not worry about the items in the body section).


The Head Section

The <head> section is where you can include meta-information about your web page. This can include the title of the page, character encoding, and links to external resources like stylesheets or scripts. The content in the head section is not displayed on the web page itself but is essential for defining how the page should be rendered and how it interacts with other resources.

All elements within the <head> tag are used to provide information about the document, but they are not displayed on the web page itself.


The Body Section

The <body> section contains the visible content of the web page. This is where you place all the text, images, links, and other media that users will see when they visit your page.

Any thing that we want to display on the web page goes within the <body> tag.


Within the <body> tag, you can use various HTML tags to structure your content. For example:

  • <h1> to <h6>: These tags are used for headings, with <h1> being the most important and <h6> the least.
  • <p>: This tag is used for paragraphs of text.
  • <a>: This tag creates hyperlinks to other web pages or resources.
  • <img>: This tag is used to embed images in the web page.
  • <div>: This is a generic container tag that can be used to group other elements together for styling or layout purposes.

  2. Write a paragraph of text using the <p> tag.
  3. Include an image using the <img> tag. You can use any image URL for this task.
