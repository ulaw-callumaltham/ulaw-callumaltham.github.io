Outputting Text to the Console

In Python, you can use the print() function to output text to the console. This is a fundamental part of programming as it allows you to display information to the user or debug your code.

Here is an example of how to use the print () function:

print("Hello, World!")
When you run this code, it will output:

Hello, World!

Try running the above code in the code editor to see how it works. You can also modify the text inside the print() function to output different messages.

For example:

print("Welcome to Python programming!")

This will output:

Welcome to Python programming!

Feel free to experiment with the print() function and see how it works with different types of data, such as numbers or variables.
